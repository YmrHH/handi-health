**修改内容**

022：Web 端//告警管理页面路由跳转修复//修复从患者异常页面（`/alert/patient`）无法跳转到硬件异常页面（`/alert/hardware`）的问题：优化 `Alerts.vue` 组件中的 `activeTab` 计算逻辑，优先使用路由名称（`route.name`）判断当前标签页类型，避免组件复用时状态不同步导致页面显示错误，确保从任何入口（侧边栏、首页跳转等）都能正确切换到硬件异常页面，解决浏览器地址栏已跳转但页面内容仍停留在患者异常页面的问题

024：Web 端//新增患者表单简化//删除新增患者弹窗中"同时为患者注册小程序账号"功能下的"确认密码"输入框，简化表单填写流程，用户只需填写小程序账号密码一次即可完成注册，同时移除相关的密码确认校验逻辑（包括"两次输入的密码不一致"的校验）和 `miniProgramPasswordConfirm` 状态变量，减少用户操作步骤  
025：Web 端//小程序注册提示文案优化//更新新增患者弹窗中"同时为患者注册小程序账号"的提示文案，从"注册小程序账号需要填写手机号和密码"改为"注册小程序账号需要填写手机号"，明确说明只需填写手机号即可完成小程序账号注册，密码由系统自动生成或使用默认规则，使提示信息与实际表单字段保持一致

# 医护端：在管患者 → 查看数据（接口扩展说明）

> 目的：在“查看数据”页面中展示 **健康指标查看 / 随访小结 / 用药与不良反应**，并支持给患者发送建议。

## 1. 获取患者数据聚合（医护端）

- **URL**：`GET /api/staff/patients/{patientId}/data`
- **鉴权**：Header `Authorization: Bearer <token>`（沿用项目现有规则）
- **Query 参数**
  - `historyLimit`：可选，历史健康指标条数（建议 7 / 30 / 90）

> - “随访小结 / 用药与不良反应”使用 `followupLatest`（若无则显示“暂无随访记录”）。

## 2. 给患者发送建议（写入患者端消息中心）

- **URL**：`POST /api/staff/patients/{patientId}/advice`
- **Body**
```json
{ "content": "建议内容..." }
```
- **返回**：标准 `code=0` 即成功

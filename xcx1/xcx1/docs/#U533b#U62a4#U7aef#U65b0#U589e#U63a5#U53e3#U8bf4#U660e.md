# 医护端新增功能接口说明（2026-01）


## 0. 在管患者列表（医护端：在管患者 Tab）

- 方法：GET
- 路径：`/api/patient/summary`
- 说明：医护端拉取“在管患者列表”。前端默认一次拉取较多（pageSize=200），并在前端做搜索过滤。

### Query（可选）
- `page`：默认 1
- `pageSize`：默认 200
- `keyword`：姓名/ID 搜索（若后端不支持可忽略）

### Response（建议）
```json
{
  "success": true,
  "code": 200,
  "message": "OK",
  "data": {
    "rows": [
      {
        "id": "p1",
        "name": "李先生",
        "age": 72,
        "riskLevel": "MID",
        "mainDisease": "高血压",
        "syndrome": "气虚"
      }
    ]
  }
}
```

本文件用于后端开发对接“医护端新增：查看患者数据、发送建议”功能。

## 1. 查看患者数据（汇总）

- 方法：GET
- 路径：`/api/staff/patients/{patientId}/data`
- 说明：医护端查看患者端同款数据（健康指标、康复任务等）的汇总接口。

### Query（可选）
- `days`：默认 7，返回最近 days 天内的指标与任务（若后端不做历史可忽略）

### Response（建议）
```json
{
  "success": true,
  "code": 200,
  "message": "OK",
  "data": {
    "profile": {
      "id": "p1",
      "name": "李先生",
      "age": 72,
      "gender": "男",
      "disease": "高血压",
      "riskLevel": "mid",
      "riskText": "中风险"
    },
    "today": {
      "sbp": 138,
      "dbp": 88,
      "hr": 72,
      "temp": 36.5,
      "weight": 70,
      "steps": 3200,
      "updatedAt": "2026-01-27 10:20:00"
    },
    "rehabTasks": [
      { "id": "rt1", "title": "步行 30 分钟", "status": "TODO", "statusText": "待完成", "date": "2026-01-27" }
    ]
  }
}
```

## 2. 发送建议（写入患者消息中心）

- 方法：POST
- 路径：`/api/staff/patients/{patientId}/advice`
- 说明：医护端直接向患者端发送建议，患者端通过 `GET /patient/messages` 可查看。

### Request
```json
{ "content": "建议控制晚餐主食并坚持每日步行 30 分钟，1 周后复测血压。" }
```

### Response
```json
{
  "success": true,
  "code": 200,
  "message": "已发送",
  "data": { "messageId": "m1001" }
}
```

## 3. 随访记录字段变更说明（前端已改）

医护端“随访记录”页面已做简化：
- 中医体征：仅保留 `tcm.pulseDesc`（脉象描述），不再提交脉位/脉速/脉形多选
- 用药信息：新增 `form.medicineName`（服用药品名称）
